# 🔥 ShreeJi Structured PrivEsc – From Shell to Root

A clean, practical, and structured **Linux Privilege Escalation helper** mapped with MITRE ATT&CK — designed for real pentesters, not just checklists.

---

## 🎯 What is this?

This is a **single-page HTML tool** that helps you:

- Run **high-impact privesc checks in the correct order**
- Focus on **signal, not noise**
- Map findings to **MITRE ATT&CK techniques**
- Quickly decide: *“What next?”*

---

## ⚡ Features

- 🧠 **Practical workflow (not theoretical)**  
- 🔍 **Search (supports fuzzy + MITRE + tactics)**  
- 🔴 **Auto-highlight exploitable findings**  
- 📌 **Decision guidance (If X → do Y)**  
- 🧾 **One-liner commands (non-root safe)**  
- ⚡ **No dependencies — just open in browser**
